## ğŸ”‘ Key Components

### AuthContext (`AuthContext.jsx`)
- **State**: `user`, `isLoggedIn`, `loading`
- **Methods**: `login()`, `register()`, `logout()`, `checkAuth()`
- **Hook**: `useAuth()` - use this in any component

### Backend Endpoints
- `GET /api/auth/verify` - Check if user is authenticated
- `POST /api/auth/user/login` - Login user
- `POST /api/auth/user/register` - Register user
- `POST /api/auth/user/logout` - Logout user

### Cookie
- **Name**: `authtoken`
- **Type**: HTTP-only (secure)
- **Contains**: JWT token with user ID
- **Set by**: Backend on login/register
- **Cleared by**: Backend on logout
